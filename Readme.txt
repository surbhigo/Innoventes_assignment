Android sdk version used:
Minimum version: 5.0.1
Maximum version :7.0

Java version:7 used

Two Github API's are used:
 
first: github library(org.eclipse.egit.github.core-2.1.5) is used to get the repository of particular person and commits and collaborators.

Second: json library (com.google.code.gson.gson-2.2.4)is used,this library is used by github internally for parsing the object.

Project Description:
In this app we are login into Github by using Github API and after successful login we get list of repository of particular username .
when a user click on a repository we get details of repository like:commit,collaborators.
